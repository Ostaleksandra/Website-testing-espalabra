# Website-Testing-Espalabra
Тестирование сайта Espalabra

В данном репозитории представлена тестовая документация для ручного тестирования сайта школы испанского языка.

Сайт для тестирования: https://espalabra.ru/

## Описание проекта

Целью проекта является проведение ручного тестирования пользовательской части сайта и проверка корректности работы основных функций.

В рамках тестирования проверялись:
- навигация по сайту
- работа ссылок
- разделы главного меню
- элементы пользовательского интерфейса

## Виды тестирования

В рамках проекта были проведены следующие виды тестирования:

- функциональное тестирование
- тестирование пользовательского интерфейса (UI)
- smoke testing
- позитивное и негативное тестирование ввода данных

Тестирование проводилось методом **черного ящика (Black Box)**.

Все проверки выполнялись **вручную (manual testing)** без использования инструментов автоматизации.

## Объект тестирования

В процессе тестирования проверялась пользовательская часть сайта:

- шапка сайта (логотип, личный кабинет, социальные сети, адрес, оплата)
- разделы главного меню
- переходы между страницами
- прокрутка страниц
- корректность ссылок и кнопок

## Тестовая документация

В данном репозитории представлены следующие артефакты тестирования:

- Test Plan — описание стратегии тестирования
- Test Cases — набор тест-кейсов для проверки функциональности
- Checklist — чек-лист для быстрой проверки основных элементов сайта

## Инструменты

В проекте использовались:

- Test Documentation
- Google Sheets
