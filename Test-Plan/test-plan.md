Введение:

Целью составления данного тест-плана является описание процесса тестирования сайта “Школа Испанского Языка”  (адрес сайта - https://espalabra.ru/). Документ позволяет получить информацию о мероприятиях по тестированию проекта.

Объект тестирования:
В процессе тестирования сайта “Школа Испанского Языка” будет тестироваться только видимая для посетителей часть сайта.
 Тестируемые элементы:
2.1 Шапка профиля: личный кабинет, логотип, ссылки на социальные сети, адрес, оплата, адрес.
2.2 Разделы главного меню: главная, школа, курсы, клубы, спецкурсы, онлайн-марафоны, об испанском, контакты.
2.3 Скроллинг страницы.
2.4 Осуществление перехода на вкладки сайта.
Не будет проведено в силу отсутствия необходимых ресурсов:
2.5 Тестирование безопасности сайта.
2.6 Нагрузочное тестирование сайта.
Виды тестирования:
3.1 Функциональное тестирование.

3.2 Тестирование пользовательского интерфейса.

3.2 Тестирование ввода e-mail и паролей:
позитивное
негативное

Порядок тестирования:

4.1 Составление чек-листа и тест-кейсов для проверки тестируемых функций сайта.

4.2 Проведение smoke testing (поверхностное тестирование основного функционала), чтобы проверить работу сайта на наличие явных ошибок.

4.3 Тестирования пользовательского интерфейса.

В силу отсутствия технической документации и ресурсов, тестирование будет проводиться по методу Черного ящика (Black box).
Тестирование будет проводиться в ручном режиме (мануальное тестирование), без использования автоматизированных систем. 

Тестовая документация:

5.1 Тест план.

5.2 Тест-кейсы.

5.3 Чек-лист.
